# Security Policy

Jotty is currently an early local-first iOS app. It does not include accounts, networking, or a production summarization backend yet.

## Reporting a Vulnerability

Please report security issues through GitHub private vulnerability reporting if it is enabled for the repository. If that is not available, contact the maintainer through GitHub and avoid posting exploit details in public issues.

Include:

- A short description of the issue
- Steps to reproduce
- Impact and affected versions, if known
- Any suggested fix or mitigation

## Scope

In scope:

- Data loss or unintended note disclosure
- Unsafe file export behavior
- Secret leakage in repository files or workflows
- Future summarization or networking issues once those features exist

Out of scope:

- General feature requests
- Issues that require jailbroken devices only
- Reports without enough detail to reproduce
